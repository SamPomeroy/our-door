Problem

Deliverable 1

Problem Framing

Cohort programs move fast and have a lot of materials: PDFs, slides, markdown files, recordings, notes, and assignments.

Students Get overwhelmed and can feel behind quickly.
Current fallback Ask classmates, Google, or use ChatGPT for direct answers.
Our angle Use the program's own materials and show repeated confusion before students fall behind.

Deliverable 1

Users + Why It Matters

Jon, Student

Gets stuck while working alone, asks privately, and receives three guided knocks: hint, curriculum reference, next thinking step.

Jane, Instructor

Sees repeated question themes and stuck points early, before quiet confusion turns into missed learning.

The goal is private learning help for Jon and clear support signals for Jane.

Deliverable 3

MVP Scope by June 2

IN

  • Canvas corpus ingested and queryable with real content
  • Backend ingestion path for admin-provided course coverage materials
  • Student chat UI
  • Three-knock guidance pattern: hint, reference, next step
  • Socratic response instead of direct answers
  • Guardrails on every response
  • Conversation logging: question, response, timestamp, topic
  • Admin view with logged conversations
  • Two hardcoded roles: student/admin
  • Docker Compose + GitHub CI

OUT

  • Individual user accounts or registration
  • Student-specific history
  • Analytics or visualizations
  • Polished file upload UI for admins
  • Multi-cohort support
